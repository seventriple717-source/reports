🧭 产业链概念速览 — 两种理解方式
光互连只做一件事：电信号 → 光 → 光纤传输 → 光 → 电信号。 T2 是唯一完成"电↔光"物理转换的核心器件，其余各层围绕"更快 / 更准 / 更便宜 / 更易用"展开。 价值链遵循一条规律：上游靠稀缺 · 中游靠算法/IP · 下游靠规模（但定价权最弱）。

T1
振膜材料 / 永磁体原料
制造器件的物理基础 — 没有 InP 晶圆就造不出 800G 激光器
T2 ★
麦克风（电→声） + 扬声器（声→电）
⭐ 唯一完成信号形态转换的器件；激光器发光、探测器收光，两者不可替代
T3
语音编解码芯片 + 智能降噪算法
DSP 负责编码/纠错；硅光 PIC 把多个光学功能集成在一块硅芯片上
T4
出厂封装好的完整听筒（含麦+芯片+外壳）
标准化"插头"，买来直接插进交换机；内部核心件来自 T2/T3
T4C

T5

⚠️ 关键洞察一：T4 是规模最大、定价权最弱的层
T4 光模块看似产业链核心，实则是精密组装商——激光器向 T2 采购（LITE/COHR 定价），DSP 向 T3 采购（MRVL 定价），剩余价值来自良率和产能管理。这正是中国厂商（中际旭创/新易盛）能快速追上的原因：组装工艺可以复制，但 InP 晶圆（T1）和 DSP 算法（T3）暂时无法国产替代。长期持仓应优先选 T1/T2/T3，而非纯组装的 T4。
⚛️ 关键洞察二：CPO 的本质是 T3 正在"吃掉"T4 的价值
传统架构：ASIC → PCB 走线(10cm 损耗) → T4 插拔模块外壳 → 光纤。CPO 架构：ASIC 与T3 硅光 PIC 光引擎直接共封装(1-3mm)，省去 T4 的机械外壳和 PCB 损耗，功耗降 75%。价值迁移结果：T4 中光引擎部分的价值从 $15 → $80+ 直接归属 T3；传统纯组装 T4 厂商价值收缩。FN（Fabrinet）专注代工制造，不承担技术路线风险，因此反而在 CPO 时代持续受益。
📡 信号链路流程图 — 每个环节在干什么

T1 基础材料
输入
In、P、Ga、As 化学元素；SiO₂（沙子）；Si 晶体
→
过程
MOCVD 外延生长；光纤拉制（2000°C）；SOI 键合
→
输出
InP/GaAs 晶圆；光纤预制棒（GLW）；SOI 晶圆
💡 为什么不可替代：III-V 族半导体（InP/GaAs）是目前唯一能高效发光的材料——硅不发光，无法制造激光器。单模光纤损耗仅 0.2dB/km，是铜缆的千分之一，是长距传输的物理基础。
T2 ★ 激光器 / 探测器（光子器件）
输入
电驱动电流（发端）/ 入射光子（收端）；InP 晶圆基底
→
过程
激光器：受激辐射产生相干光；探测器：光电效应将光子→电流
→
输出
相干光束（1310/1550nm）/ 微弱模拟电流
💡 产业链唯一的"变身"器件：光纤只传光不传电，所有通信必须经过 T2 完成电↔光转换。硅无法发光，必须用 InP III-V 材料，全球只有少数厂商掌握批量制造工艺（COHR/LITE/MTSI）。
T3A 硅光平台（SiPh PIC）
输入
T2 激光器的连续光（CW light）+ 高速电信号
→
过程
MZM 调制器将数据"写入"光波；多路 WDM 合波；集成分束/合束无源器件
→
输出
承载数据的调制光信号（PAM4 / QPSK / QAM16）
💡 为什么重要：InP 光芯片昂贵（$50+/片），SiPh 用 CMOS 晶圆厂（TSMC/GFS）制造，成本降 1/3。CPO 时代 PIC 价值从 $15→$80+ 每模块，T3A 是"吃掉 T4 价值"的核心路径。
T3B DSP / TIA / SerDes 芯片
输入
高速原始比特流（来自 ASIC）/ 微弱模拟光电流（来自探测器）
→
过程
FEC 前向纠错；PAM4/QAM 编解码；SerDes 并串转换；TIA 跨阻放大
→
输出
可靠低误码数字信号（BER <10⁻¹⁵）；SerDes 标准电口
💡 AI 互连的"大脑"：800G 原始信号经 100m 光纤后误码率高到无法使用；DSP 用算法从噪声中恢复信号。MRVL 外部市占率 >60% 来自数代 IP 积累和 3–5 年客户 co-design 周期，壁垒极深。
T4A 数据中心光模块（400G / 800G / 1.6T）
输入
T2 激光器裸片 + T3A SiPh PIC + T3B DSP 芯片（散件）
→
过程
精密组装封装；sub-μm 光路对准；热管理；DMI 接口；MSA 标准认证
→
输出
QSFP-DD / OSFP 标准插拔模块（买来直接插进交换机）
💡 本质是精密组装：核心器件来自 T2/T3，T4 仅提供标准化接口和封装良率。定价权最弱——中国厂商（中际旭创/新易盛）已能复制，ASP 持续承压。CPO 2026 年商用后将部分替代此形态。
T4B 相干光模块（ZR / ZR+ / 长途相干）
输入
窄线宽 CW 激光源 + 先进相干 DSP（MRVL/COHR）+ ICR 相干接收机
→
过程
QAM16/64 相位+幅度双编码；ICR 干涉感知光的相位；DSP 色散补偿均衡
→
输出
400ZR / ZR+ CFP2 / QSFP-DD（$800–$2000+ ASP，无中继 80km+）
💡 vs T4A 区别：T4A 用简单 PAM4（强度编码），只能传 2km；T4B 相干调制感知光的相位，同样带宽下传 80km+，频谱效率 10× 提升。ICR + 先进 DSP 组合是中国暂时无法追赶的壁垒。
T4C 连接器 / Cage / AOC 线缆
输入
T4A/T4B 光模块 + 精密金属/塑料机械件 + 高频电气触点
→
过程
OSFP Cage 标准化机械接口；高频信号完整性；散热管理；AOC 内嵌收发器
→
输出
OSFP / QSFP-DD Cage 总成；AOC 主动光缆；高密背板连接器
💡 最被忽视但最确定的环节：AI DC 每台 512 端口交换机需 512 个 OSFP Cage；端口密度每代翻倍，Cage 需求刚性正比增长。APH/TEL 供给高度集中，与光模块价格战完全无关，确定性极高。
T5A WDM / 相干传输系统
输入
多路 T4B 相干模块输出（每路不同波长，96 路 C-Band）
→
过程
WDM 复用器合 96 波入单根光纤；EDFA 掺铒光纤放大（全光，无 O-E-O）；NMS 网管
→
输出
单对光纤承载 96×400G = 38.4Tbps；完整运营商 / DCI 骨干传输网
💡 价值在"系统集成"：WDM 让一根光纤同时传 96 个波长，省去 95 对光纤；EDFA 全光放大无需 O-E-O 转换，功耗降 70%。CIEN 的 WaveLogic DSP 领先同业约 2 年，是其毛利率 50%+ 护城河的核心。
T5B ROADM / OXC 光交换
输入
多方向 WDM 信号（东西南北各方向光纤汇聚）
→
过程
WSS 波长选择开关（LCOS 芯片）按波长路由；全光域不经 O-E-O；可远程重配置
→
输出
各波长按需路由到不同出口方向（整条 100Gbps 光管道级调度）
💡 vs 电路由器：电路由器对每个包做 O-E-O（功耗巨大）；ROADM 把整条波长"一刀切"路由，光子全程不变为电子，功耗降约 70%。AI 工厂能耗压力是 ROADM 替代电交换的长期驱动力。LITE 的 WSS 市占率 >50%。
T5C 以太网交换机 / AI 集群网络集成
输入
数百–数千台 GPU 服务器的 800G 光端口；BGP / ECMP 路由配置
→
过程
L2/L3 Ethernet 包交换；Clos 网络拓扑；超低延迟（<1μs）；512×800G 端口密度
→
输出
数万 GPU 全互连 AI 集群；"超级计算机"级分布式训练网络
💡 光模块最大的单体需求方：ANET 每卖出一台 512 端口交换机，直接带动 512 个 800G 光模块出货（约 $20 万模块价值/台）。2026 年 ANET+AVGO 联合推 CPO 交换机，将从"买模块"转变为"卖含光引擎的整机"。
